----- Database - בסיס נתונים -----
	מאגר של נתונים המכיל את המידע בטבלאות
	לכל טבלה יש עמודות
	והמידע הוא בעצם שורות בטבלה
	
	כדי לדבר עם הבסיס נתונים משתמשים בשפת שאילתות הנקראת SQL
	בעזרתה אפשר: להוסיף מידע לבסיס הנתונים, למחוק, לעדכן ולשלוף
	
	מסד הנתונים בו נשתמש הוא: SqlServer
	
	

----- קטעי קוד נפוצים -----
- העברה מדף אחד לדף שני:
	Response.Redirect("Default.aspx");
	
	
- מבנה הפרוייקט
	- קבצי ה aspx	-	הדפים באתר
		הדפים כתובים בשפת HTML
		בנוסף יש בהם פקדים מיוחדים של asp.net
		הפקדים נראים ככה: <asp:XXX
		ניתן לשלב אף קוד בתוך דפים אלו בתוך התגיות: <%...C# code...%>
	- לכל דף aspx
		קיים דף שנקרא code behind
		דפים אלו מסתיימים ב aspx.cs
		בדף זה נרשום את הקוד שקשור לעמוד
		למשל: מה יקרא שהעמוד נטען, מה יקרה שנלחץ על כפתור וכו
	- תיקיות בפרוייקט
		- Styles	-	שם נשמור את קבצי ה css
			אלו קבצי העיצוב של האתר
		- Images 	-	שם נשמור את התמונות שאנו טוענים באתר
		- App_Date	-	שם יושב בסיס הנתונים
			קובץ בסיס הנתונים הוא בסיומת mdf
			אנחנו משתמשים בבסיס נתונים הנקרא: Sql Server
		- App_Code	-	שם יושב הקוד c# השונה באתר
		
- תגים שונים
	- title 	-	קובע הטקס שיופיע בטאב של העמוד
	- link		-	תג המשמש לקישור קובץ css לעמוד html
	
- פקדי aspx
	לכל פקד נצטרך לתת שם - ID
	בעזרת השם נוכל לקשר את הפקד לקוד
		- label 	-	פקד המשמש לכתיבת טקסט
		- TextBox	-	פקד המשתמש לקליטת טקסט מהמשתמש
		- Button 	-	פקד המשמש ככפתור
	
	
- פונקציות מיוחדות ב aspx
	- Page_Load		-	פונקציה המתבצעת כאשר העמוד נטען
	- XXX_Click		- 	פונקציה המתבצעת כאשר לוחצים על כפתור
	
	
Session	-	משתנה מיוחד שנשמר בין כל דפי האתר, כל עוד המשתמש מחובר
	נהוג לשמור שם מידע כמו:
		- סוג המשתמש - האם רגיל או מנהל
		- שם המשתמש
		- פרטי המשתמש
		- עגלת קניות
		
	יצירת משתנה בתוך ה session:
		Session["age"] = 20;
		Session["name"] = "Yossi";
		Session["height"] = 1.73;
