----- טכנולוגיות שנשתמש בבניית האתר -----
- HTML				-	בניית דפי האתר
- CSS				-	עיצוב דפי האתר
- ASP.Net + C#		- 	פונקציונליות בצד השרת
- Database			-	בסיס נתונים
- SQL 				- 	השפה שבה הקוד מדבר עם הבסיס נתונים - שפת שאילתות
	סוגי שאילתות ב SQL:
		SELECT 	-	שאילתה שמטרתה לקבל מידע מבסיס הנתונים
		INSERT	-	שאילתה שמטרתה להוסיף מידע לבסיס הנתונים
		DELETE	-	שאילתה שמטרתה למחוק מידע מבסיס הנתונים
		UPDATE	-	שאילתה שמטרתה לעדכן מידע שבבסיס הנתונים
- Javascript		-	פונקציונליות בצד הלקוח





- משימות
1) יצירת הדאטה בייס יחד עם הטבלאות
2) לגרום להרשמה לעבוד - לאחר הכנסת נתונים תקינים, להוסיף רשומה לטבלת המשתמשים
3) לגרום להתחברות לעבוד - בדיקה אם השם משתמש והסיסמה נכונים, במידה לגרום למשתמש להיות מחובר ולהעביר אותו לדף הבית
4) יצירת יכולת התנתקות
5) בדף הבית, אם המשתמש מחובר נרשום ברוך הבא עם השם שלו
6) יצירת עמוד עם פרטי המשתמש שבהם הוא רואה את פרטיו ושם הוא יכול לשנות סיסמה
7) יצירת עמוד עבור המנהל שבו הוא יוכל לראות את פרטי כל המשתמשים באתר
8) בנוסף יהיה לו שם אופציה למחיקה של משתמש מסוים
9) ליצור יכולת להוסיף מוצרים לסל
10) לשמור את המוצרים שבסל של כל משתמש בבסיס הנתונים כך שלא ימחק לאחר התנתקות
11) ליצור אפשרות של הזמנת המוצרים שבסל + שמירת ההזמנה בבסיס הנתונים
